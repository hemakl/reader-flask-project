Book Wishlist RESTful API

Welcome to the documentation for the Book Wishlist API.

RESTful Routing - URL and HTTP verb patterns

URL Path GET POST PUT DELETE
/users list users create new user - -
/users/<user_id> get user - update user delete user
/users/<user_id>/books list books in wishlist create book - delete books in wishlist
/users/<user_id>/books/<book_id> get book - update book delete book in wishlist
